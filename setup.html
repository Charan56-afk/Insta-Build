Enable Real Google Sign-In

Follow these steps to connect your Google account — exactly like Instagram & Facebook. Takes ~2 minutes.

✅ Google Sign-In is Ready!

Your Client ID has been saved. Real Google login is now active in InstaBuild.


← Go to InstaBuild & Sign In
1
Open Google Cloud Console
We need to create a free Google OAuth 2.0 Client ID. This tells Google that InstaBuild on your computer is allowed to use Google Sign-In.
  
2
Create OAuth 2.0 Client ID
In Google Cloud Console, do the following:
Click APIs & Services → Credentials in the left sidebar
Click + CREATE CREDENTIALS → OAuth 2.0 Client IDs
Application type: Web application
Under Authorized JavaScript origins, click "+ ADD URI" and paste:
http://localhost:3001 click to copy
⚡ Tip: If Google asks to configure the OAuth consent screen first, choose External, fill in your App name (e.g. "InstaBuild"), your email, then save and go back to create credentials.
3
Paste your Client ID
Copy the Client ID from Google Cloud Console. It looks like: 123456789-xxxx.apps.googleusercontent.com
